Projeto Intellimen

O projeto intellimen é um desafio de 53 semanas (1 ano) para homens que desejam se tornar melhores não só para si, mas para todos ao seu redor, tanto em casa, como no trabalho, escola, faculdade, igreja, etc.

Por isso é necessário comprometimento com os desafios, realizando cada um deles com o pensamento de que no final deles você será um homem melhor.

Por exemplo: esse site foi criado para completar o desafio 20, mas apesar das dificuldades conseguir completá-lo.
